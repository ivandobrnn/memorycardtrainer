# Memory Card Trainer Web

Статическая PWA-версия тренажера находится в этой папке.

Порядок карт фиксированный, по схеме MemorizeDeck с картинки: `1 = 4♣`, `2 = 2♥`, ..., `52 = 9♦`.
Режимы обучения: `1-13`, `14-26`, `1-26`, `27-39`, `1-39`, `40-52`, `1-52`.

Локальный запуск:

```powershell
cd web
.\serve-local.ps1
```

После этого открой `http://localhost:4173/`.

Установка на телефон:

1. Опубликуй содержимое папки `web/` на HTTPS-хостинге, например GitHub Pages, Netlify или любом своем сайте.
2. Открой адрес на телефоне в Chrome, Edge или Safari.
3. Выбери установку приложения в меню браузера.
4. После первого запуска приложение, колода и статистика доступны без интернета.

Важно: service worker для офлайн-режима работает только на HTTPS или на `localhost`.
